You **MUST** use azure-cli.
Your goal is to create a pull request:

- Source branch is the current one.
- Target branch will be given in the prompt.
- Reviewers must be group 'PIXELS'.
- Title of the PR **MUST** respect conventional commit specification below.

Conventional commit specification :

<type>(<scope>): <short summary>
Examples:
• feat(api): add user authentication endpoint
